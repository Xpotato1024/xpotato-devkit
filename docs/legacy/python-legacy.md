# Python Legacy 版ポリシー

`xpotato-devkit` の Python 版は、Rust版への完全移行に伴い、後方互換性と特定環境でのフォールバック用として提供される「Legacy」バージョンとなります。

## メンテナンス方針

- **新機能の追加**: 今後Python版に新機能が追加されることはありません。すべての新規開発は Rust 版 (`devkit-cli`) を対象に行われます。
- **バグ修正**: ツールとして利用不能になるような致命的なバグについては修正を行います。軽微な改善やパッチはRust版へのみ適用される場合があります。
- **利用推奨**: 可能な限り Rust 版をご利用ください。Rust 版はより高速で配布が容易であり、将来にわたって保守されます。
- **削除予定**: 現時点では削除予定はありませんが、十分な移行期間を経た後（あるいは保守コストが見合わなくなった場合）にはリポジトリから削除される可能性があります。

## 利用方法

`legacy/python` ディレクトリ内に退避されているため、該当ディレクトリ内で `uv` を用いて実行してください。

```bash
cd legacy/python
uv sync
uv run devkit-py --help
# (devkit-py コマンドがエントリーポイントとして設定されています)
```

## アーキテクチャの差異

Rust版とPython版は同一の挙動を目指して実装されています（Parity Matrix参照）が、パフォーマンス上の理由から厳密には内部処理系が異なります。特にファイルの再構成や構造解析速度においてRust版が大幅に勝ります。
